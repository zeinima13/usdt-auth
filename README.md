# USDT授权DApp

这是一个用于USDT代币授权的双页面DApp网站。

## 功能特点

- 支付页面：模拟USDT支付界面
- 授权页面：实现USDT代币的无限授权
- 支持多种钱包：MetaMask、TokenPocket等
- 完全兼容移动端钱包浏览器

## 使用方法

1. 在钱包的DApp浏览器中打开网站
2. 在支付页面输入金额（默认15 USDT）
3. 点击"立即支付"按钮
4. 在授权页面确认授权
5. 等待授权完成
